# React Webpack Boilerplate
Kickstart your project with thiz webpack boilerplate, all set up and ready to go!

- Webpack
- React
- React Router
- PostCSS / CSS Modules+

## Quick start?
1. **yarn**
2. **yarn start**
3. Browser will open your website automagically

## Production?
1. **yarn build**
2. Look at your **dist/** folder
